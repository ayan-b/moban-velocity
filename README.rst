moban-velocity
================================================================================

With `airspeed` for Python 3, this library allows moban users to have velocity
template in their next documentation endeavour. 

Installation
================================================================================

You can get it:

.. code-block:: bash

    $ git clone https://github.com/moremoban/moban-velocity.git
    $ cd moban-velocity
    $ python setup.py install
